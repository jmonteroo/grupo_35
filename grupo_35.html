Fortnite

Fortnite - Juego de ritmo y accion, multijugador en linea que combina el juego del ultimo hombre en pie con los elementos de supervivencia y exploracion

Comparacion con otro juego

Pubg. Diferencias con el juego principal: Pubg es mas realista y tactico, mientras que fortnite mas rapido. Fortnite tiene construccion de estructuras, pubg no. Pubg suele tener graficos mas realistas

  • Diferencia 1: El genero del pubg es battle royale realista mientras que el del fortnite es battle royale arcade
  • Diferencia 2: El estilo del pubg tiene graficos mas serios, armas realistas y fisicas mas cercanas a la realidad mientras que fortntie tiene graficos coloridos y caricaturisticos
  • Diferencia 3: El pubg requiere requisitos recomendados mas altos que el fortnite para un mejor rendimiento esperado

Requisitos del sistema

Fortnite

  • Minimos: CPU: Core I3-3225 3.3Ghz, GPU: Intel HD 4000 o equivalente, RAM: 4GB, SO: windows 7/8, almacenamiento: 70GB
  • Recomendados: CPU: Core i5-7300U 3.5GHZ o superior, CPU: Nvidia gtx 960, ram: 8GB, SO: windows 10, almacemantiento: 70GB

Pubg

  • Minimos: CPU: Intel Core i5-4430, CPU: Nvidia geforce gtx 960 2gb, RAM: 8GB, SO: windows 7, almacenamiento: 40GB
  • Recomendados: CPU: Intel Core i5-660k, CPU: Nvidia geforce gtx 1060, RAM: 16GB, SO: windows 10, almacenamiento: 50GB
